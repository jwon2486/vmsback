[기획안] S&SYS 방문객 출입 관리 시스템(VMS) 고도화 및 개선 요약 (최종 확정안)
1. 임직원 대시보드 UI/UX 단순화 (부서 담당자 기능 폐지) 🆕

부서 예약 조회 폐지: 기존 부서 담당자(Level 2) 권한 및 '부서원 예약 열람' 탭을 완전히 폐지하여 시스템 구조를 단순화합니다.

직관적인 2분할 화면 고정: 모든 일반 임직원은 불필요한 기능 없이 [새 방문객 예약]과 [나의 예약] 구역으로만 나뉜 깔끔한 2분할 대시보드를 사용하게 됩니다.

대리 신청 권한 격상: 타 직원의 방문객을 대신 등록해 주는 '대리 신청' 기능은 일반 임직원 화면에서 제거하고, 최고 관리자(Level 3) 전용 기능으로 제한하여 관리 체계를 강화합니다.

2. 방문객 등록 절차 개편 및 수집 정보 현행화

입력 폼 최적화 및 정보 수집: 방문객은 로비 도착 시 다음의 필수/선택 정보를 직접 기입합니다.

방문객 정보: 본인 이름, 소속 회사명, 본인 연락처, 차량 번호(미지참 시 '없음')

방문 목적: 회의/미팅, 제품 납품, 기타 등

담당자 정보: 사내 방문 담당자 성명 (단순 텍스트 입력)

임직원 명단 블라인드 처리: 기존의 사내 직원명 실시간 검색(자동완성) 기능을 완전히 제거하여 외부인에게 사내 조직도가 노출되는 것을 원천 차단합니다.

백엔드 블라인드 매칭: 방문객이 텍스트로 입력한 담당자 이름을 서버가 내부 데이터베이스와 대조하여 사번(ID)을 암묵적으로 매칭합니다.

3. 지사별 가상 분할(Virtual Multi-tenancy) 및 QR코드(BYOD) 도입

URL 파라미터 기반 지역 격리: 테크센터, 에코센터 등 지사별로 별도의 시스템을 구축하지 않고, 접속 URL에 지역 파라미터(예: ?region=에코센터)를 부여하여 단일 코드로 시스템을 가상 분할합니다.

개인 스마트폰(BYOD) 활용: 공용 로비 키오스크 대신, 각 지역 파라미터가 담긴 QR코드를 로비에 비치합니다. 방문객은 본인의 스마트폰으로 스캔하여 시스템에 접속 및 등록을 진행합니다.

지역 기반 1차 필터링: 백엔드에서 담당자를 검색할 때, 방문객이 접속한 QR코드의 지역(Region) 내에서 우선 검색하여 동명이인 발생 확률을 대폭 낮춥니다.

4. 보안실(Level 4) 중심의 대면 승인 및 출입 통제 프로세스 확립

보안 전용 권한 신설: 시스템 내에 보안실(경비실) 담당자를 위한 최고 보안 권한인 'Level 4'를 신설합니다.

방문객 상태값(Status) 도입: 방문객이 스마트폰으로 등록/퇴실을 요청하더라도 즉시 입퇴실이 완료되지 않으며, DB상에 '입실대기' 또는 '퇴실대기' 상태로 임시 저장됩니다.

최종 승인 시점의 시간 기록 (핵심): 보안실 요원(Level 4)이 방문객을 대면 확인한 후, 시스템에서 직접 [승인] 버튼을 클릭하는 바로 그 순간의 KST(한국 표준시) 서버 시간이 방문객의 최종 시간으로 DB에 확정 기록됩니다.

5. 예외 상황(동명이인/출장 등)의 스마트 자동 이관

복잡한 추가 인증 제거: 방문객에게 담당자의 소속 부서나 연락처를 묻는 복잡한 과정을 생략하여 사용자 경험(UX)을 극대화합니다.

보안실 수동 확인(Fallback) 프로세스: 시스템이 전사 조회를 통해 담당자 1명을 완벽히 특정하지 못한 경우(동명이인 충돌, 타 지사 출장 등), 해당 방문객의 담당자 ID를 임시 보류 상태(guard_pending)로 지정합니다.

안내 메시지 통일: 예외 상황이 발생해도 방문객에게는 오류 대신 "안내 데스크(경비실)로 이동해 주세요"라는 정상적인 메시지를 출력하고, 최종 식별 및 담당자 매칭 책임은 보안 요원에게 넘깁니다.

6. 보안실 전용 맞춤형 대시보드 통합

단일 페이지 동적 렌더링: 별도의 페이지를 만들지 않고, 기존 임직원 화면(/emp)에서 로그인한 직원의 권한이 Level 4(보안실)일 경우 보안실 전용 대시보드가 자동으로 열리도록 구현합니다.

원클릭 관제: 보안실 대시보드에는 모든 방문객의 내역, 차량 번호, 연락처가 표기되며 직관적으로 입실/퇴실 버튼을 눌러 승인할 수 있습니다.

지역별 대기열 관제: 테크센터 보안 요원이 로그인하면 테크센터의 내역만, 에코센터 보안 요원이 로그인하면 에코센터의 내역만 화면에 출력되어 지사별 보안 업무의 독립성을 보장합니다.
